Hashing is a technique for changing data content into unique data that has the same length.

sha256 is a type of cryptographic hash algorithm that converts input data into a 256-bit sequence of random alphanumeric data.

SHA-256 generates a unique fingerprint of the firmware file. Any modification, even a single byte, produces a completely different hash value. This allows the system to detect tampering and provides a compact representation of the firmware that can later be digitally signed using the developer's private key.
