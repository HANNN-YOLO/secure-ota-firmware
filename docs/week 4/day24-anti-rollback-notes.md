Today I'm going to learn about Anti-Rollback.

So, about rollback. Rollback is a downgrade by an attacker because the attacker sees a different version semantics, so the attacker thinks that if they attack by downgrading, they will definitely find vulnerabilities from this old version that are still valid.

This is where it's called anti-rollback. Anti-rollback is a defense to prevent downgrades by attackers. Anti-rollback has the logic to not downgrade if the attacker wants to force a downgrade.
